With this tool you can select elements.

## Konfiguration

| Objekt |          Standard          | Beschreibung                                                                             |
| ------:|:--------------------------:|:---------------------------------------------------------------------------------------- |
|  achse | (horizontal oder vertikal) | Richtung, wo der Raum hinzugefügt werden soll. Nur beim Hinzufügen des Werkzeugs wählbar |
