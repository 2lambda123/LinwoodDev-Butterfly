With this tool you can select elements.

## Configuración

| Propiedad |       Por defecto       | Descripción                                                                           |
| ---------:|:-----------------------:|:------------------------------------------------------------------------------------- |
|       eje | (horizontal o vertical) | Dirección donde debe añadirse el espacio. Sólo seleccionable al añadir la herramienta |
