## Johdanto

![Alue](area.png)

Alueet ovat keino rajoittaa kankaan kokoa. You can create areas by using the [area tool](tools/area.md). Jos napsautat taustaa, voit avata alueen luettelon valintaikkunan käyttämällä konteksti-valikosta.

## Alueiden luominen ja poistaminen

Use the [area tool](tools/area.md) to create an area. Alueet eivät voi olla päällekkäisiä. Kun olet luonut alueen, sinun täytyy nimetä se.

Klikkaa aluetta, jos haluat muokata sitä. Voit myös käyttää alueen valintaikkunaa.

Sinulla on muutamia vaihtoehtoja tässä valikossa:

* `Syötä alue`: Syötä alue. Alueen ulkopuolelle ei voi maalata. Exit the area by clicking on the background with the area tool.
* `Nimi`: Muuta nykyisen alueen nimeä
* `Vie`: Vie tämä osa kankaalle kuvana
* `Poista`: Poista tämä alue
