Avec cet outil, vous pouvez sélectionner des éléments.

## Configuration

| Propriété | Par défaut | Libellé                                                        |
| ---------:|:----------:|:-------------------------------------------------------------- |
|      mode | rectangle  | Mode de sélection. Les données valides sont rectangle et lasso |
