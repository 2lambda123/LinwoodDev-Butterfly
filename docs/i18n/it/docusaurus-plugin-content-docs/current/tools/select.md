Con questo strumento è possibile selezionare gli elementi.

## Configurazione

| Proprietà | Predefinito | Descrizione                                           |
| ---------:|:-----------:|:----------------------------------------------------- |
|  modalità | rettangolo  | Modalità di selezione. Valido sono rettangolo e lazzo |
