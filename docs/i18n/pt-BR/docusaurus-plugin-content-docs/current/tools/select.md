Com esta ferramenta você pode selecionar elementos.

## Configuração

| Propriedade |  Padrão   | Descrição:                                       |
| -----------:|:---------:|:------------------------------------------------ |
|        Modo | retângulo | Modo de seleção. Válido são o retângulo e o laço |
