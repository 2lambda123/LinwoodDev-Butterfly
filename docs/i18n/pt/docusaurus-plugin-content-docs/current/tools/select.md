## Configuração

| Propriedade |          Padrão          | Descrição:                                                                                   |
| -----------:|:------------------------:|:-------------------------------------------------------------------------------------------- |
|        eixo | (horizontal ou vertical) | Direção onde o espaço deve ser adicionado. Apenas selecionável quando adicionar a ferramenta |
