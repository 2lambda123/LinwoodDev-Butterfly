## Configurare

| Proprietate |         Implicit         | Descriere                                                                        |
| -----------:|:------------------------:|:-------------------------------------------------------------------------------- |
|         axă | (orizontal sau vertical) | Direcția în care ar trebui adăugat spațiul. Selectabil doar la adăugarea uneltei |
