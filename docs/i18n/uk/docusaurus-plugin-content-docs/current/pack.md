title: "Набір"
sidebar_position: 2
---

Пакет є набором компонентів, які можуть бути використані в штамп. Ви можете створити власний пакет або імпортувати наявний. Для цього клацніть на лівому верхньому лівому куті та натисніть на "Пакунки".

## Властивості

|  Ім'я | Опис          |
| -----:|:------------- |
|  Ім'я | Ім'я пакунку  |
| Автор | Автор пакунку |
|  Опис | Опис набору   |
