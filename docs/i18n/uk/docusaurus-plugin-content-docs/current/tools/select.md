За допомогою цього інструменту можна вибрати елементи.

## Конфігурація

| Властивість |   Типово    | Опис                                       |
| -----------:|:-----------:|:------------------------------------------ |
|       режим | прямокутник | Режим вибору. Дійсно є прямокутник і лассо |
