Можете починати застосунок з аргументами.

`butterfly.exe <Path>`

Шлях до файлу, який потрібно відкрити.

## Опції

|          Опція | Опис                                                       |
| --------------:|:---------------------------------------------------------- |
| `--path`, `-р` | Виберіть каталог даних для зберігання всіх даних метелика. |
