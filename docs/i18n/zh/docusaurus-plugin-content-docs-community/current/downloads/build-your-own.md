1. 安装 git 和 flotter (测试版)
2. 复制资源库
3. 导航到应用目录
4. 使用流体工具编译应用程序
   * `通风器构建apk`
   * `流体构建appbundle`
   * `流体构建网络`
   * `流体构建Linux`
   * `通风器构建窗口`
   * `flutter build ios --release --no-codesign`\
     after that, create a folder named "Payload", copy Runner.app into it and zip the payload folder. 然后将“.zip”重命名为“.ipa”。
5. 编译的文件在构建目录
