工具可以编辑文档。

使用侧边栏了解更多关于工具的信息。

点击加号图标创建新工具。 通过拖动和点击改变工具顺序来改变工具顺序。 如果您点击选中的工具，您可以配置它。

工具名称将显示为工具上悬停的工具提示。
